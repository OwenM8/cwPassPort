Introduction

PassPort is a specialized SDK designed to handle the complexity of the WebAuthn API. It allows you to implement passwordless login without worrying about binary buffers, challenges, or hardware compatibility.

Installation

Add the PassPort client-side library to your project via our CDN.

<script src="https://cdn.passport-auth.io/v1/sdk.js"></script>

Registration

To create a new passkey, pass a user object containing a unique identifier and an email address.

const user = {
  id: "user_9921",
  email: "jane.doe@example.com"
};

const credential = await passport.register(user);

Once successful, the SDK returns a PublicKeyCredential object which you should store on your server.

Authentication

To sign a user in, simply call the authenticate method. The browser will automatically find valid passkeys for your domain.

try {
  const session = await passport.authenticate();
  console.log("User verified:", session.user.id);
} catch (err) {
  handleError(err);
}
