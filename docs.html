Installation

Paste this script tag into your <head>. This initializes the passport object on your window.

<script src="https://yourdomain.com/sdk.js"></script>

Registering a User

To create a passkey, call the register method. Ensure you pass a valid email string.


// Triggered by a button click on your site
try {
  const credential = await passport.register("user@email.com");
  console.log("Save this to your DB:", credential);
} catch (err) {
  console.error("Auth failed", err);
}

Authenticating

Login requires no arguments. The browser will automatically prompt the user for their saved passkey.


const assertion = await passport.login();
// Verify assertion.response.signature on your backend
